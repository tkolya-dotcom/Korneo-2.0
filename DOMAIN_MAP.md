# DOMAIN MAP — Korneo

> Карта предметной области проекта Korneo. Используется как единый источник правды при миграции на мобильный клиент.

## Роли пользователей

| Роль | Код | Описание |
|------|-----|----------|
| Рабочий | `worker` | Выполняет задачи, заполняет АВР, ведёт табель |
| Инженер | `engineer` | Создаёт задачи, подписывает АВР, контролирует объекты |
| Менеджер | `manager` | Управляет проектами, согласует заявки |
| Зам. руководителя | `deputy_head` | Расширенный доступ, согласование |
| Администратор | `admin` | Полный доступ, управление пользователями |

## Статусы задач

| Статус | Код | Описание |
|--------|-----|----------|
| Новая | `new` | Задача создана, не взята в работу |
| В работе | `in_progress` | Исполнитель работает |
| На паузе | `on_hold` | Временно приостановлена |
| Выполнена | `completed` | Работы завершены |
| Архив | `archived` | Закрыта/архивирована |

## Статусы АВР

| Статус | Код | Описание |
|--------|-----|----------|
| Черновик | `draft` | Создан, не отправлен |
| На подписи | `pending_signature` | Ожидает подписи инженера |
| Подписан | `signed` | Подписан всеми сторонами |
| Отклонён | `rejected` | Отправлен на доработку |

## Статусы заявок на закупку

| Статус | Код | Описание |
|--------|-----|----------|
| Черновик | `draft` | Создаётся |
| На согласовании | `pending` | Ожидает менеджера |
| Согласована | `approved` | Одобрена |
| Отклонена | `rejected` | Отказано |
| Закрыта | `closed` | Выполнена |

## Ключевые сценарии (user flows)

### Рабочий
1. Авторизация → Dashboard → Мои задачи
2. Задача → Детали → Комментарий / Вложение
3. Заполнить АВР → Отправить на подпись
4. Заявка на материалы → Создать → Отправить
5. Мессенджер → Чат с инженером

### Инженер
1. Авторизация → Dashboard → Объекты
2. Создать задачу → Назначить рабочего
3. АВР → Подписать / Отклонить
4. Карта → Посмотреть объекты
5. Согласовать заявку

### Менеджер / Зам. руководителя
1. Авторизация → Dashboard → Проекты
2. Просмотр табеля
3. Согласование заявок на закупку
4. Отчёты по объектам

## Технологический стек (Mobile)

- **Framework**: Expo SDK 51+ (React Native)
- **Navigation**: Expo Router (file-based)
- **State**: Zustand + React Query
- **Backend**: Supabase (shared с PWA)
- **Push**: Expo Notifications + FCM
- **Maps**: react-native-maps + Mapbox GL
- **UI**: NativeWind (Tailwind for RN)
- **Build**: EAS Build + EAS Submit
